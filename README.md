# RAKUFULL 🎵

> 会計記帳を、楽しく。楽に。

## これは何？

クラウド会計ソフトの入力画面に、サウンドエフェクト・テーマカラー・環境音を追加する **非公式Chrome拡張機能**です。

DX化・自動化がトレンドの今、あえて「入力体験そのものを楽しくする」というコンセプトで作りました。

会計事務所に勤める中で、個人事業主の方々が会計入力を苦手とする場面を多く見てきました。正確さを重視するあまり無機質になりがちな会計ソフトに、音・色・エフェクトという感覚的な演出を加えることで、めんどうな記帳の時間が少しでも癒しの時間になればと思い開発しました。

## 機能

- 🎨 **テーマカラー変更** — パステル6色（ブルー・ティール・ローズ・パープル・アンバー・ナイト）
- 🎹 **入力SE** — キーを押すたびに心地よい音（ピアノ・マリンバ・ポップ・8bit）
- 🌿 **環境音** — 波・焚火・雨をBGMとして流せる
- ✨ **登録エフェクト** — 登録ボタンを押すとコイン＆コンフェッティが舞う
- 🖱️ **コンソールパネル** — 画面上を自由に動かせる設定パネル

## 対応ソフト

- やよいの白色申告 オンライン
- やよいの青色申告 オンライン
- freee会計

## インストール方法

1. [最新版のZIPをダウンロード](https://github.com/torineco/rakufull/releases)
2. 解凍する
3. Chromeで `chrome://extensions/` を開く
4. 右上「**デベロッパーモード**」をオンにする
5. 「**パッケージ化されていない拡張機能を読み込む**」→ 解凍したフォルダを選択
